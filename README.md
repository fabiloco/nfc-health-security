## Description

Backend application to authorize NFC tokens based on ethereum blockchain

## Installation

```bash
yarn install
```

## Setup docker mysql database

```bash
docker compose up -d
```

## Running the app

```bash
# development
yarn run start

# watch mode
yarn run start:dev

# production mode
yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
